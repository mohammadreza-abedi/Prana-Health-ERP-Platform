بریف پروژه: دستیار هوشمند پرانا

هدف کلان: توسعه یک پلتفرم هوشمند سلامت شغلی، سلامت فردی و ارتقاء منابع انسانی برای کارمندان، مدیران HR/HSE و عموم مردم.

استایل برند: ترکیب Glassmorphism و مینیمالیسم مدرن؛ رنگ‌های برند پرانا (Tiffany Blue، Navy Blue، Aqua، Yellow Accent).

معماری:

بک‌اند: Flask یا FastAPI

فرانت‌اند: TailwindCSS + Framer Motion + Alpine.js

دیتابیس: PostgreSQL (رسمی) + Redis (کش)

API First توسعه‌یافته.

ساختار پروژه: ماژولار و Headless؛ هر بخش مستقل و API-Driven.

ویژگی‌های کلیدی: داشبورد سلامت زنده، گیمیفیکیشن پیشرفته، چت‌بات هوشمند، اسیستنت داخلی پروژه‌ها.

ورود کاربران: سیستم OAuth2 (گوگل، گیت‌هاب) + احراز هویت ۲ مرحله‌ای.

داشبوردها:

برای کاربران عادی: خدمات سلامت و مشاوره هوشمند

برای مدیران HR/HSE: ابزارهای تحلیلی تخصصی و گزارش‌های سلامت سازمانی

برای پرسنل داخلی پرانا: پنل مدیریت خدمات و پروژه‌ها (مخفی).

تجربه کاربری (UX):

بسیار روان، انیمیشن‌های ظریف، فرم‌های تعاملی، Drag & Drop، سیستم روشن/تاریک.

ماژول‌های ویژه: چالش‌های سلامت، مسابقات، سیستم امتیاز XP، باشگاه کاربری، تحلیل رفتار.

ارتباطات: چت‌بات‌های تخصصی سلامت، فیتنس، تغذیه، روانشناسی؛ اتصال بلادرنگ WebSocket.

پلتفرم‌سازی: PWA کامل، قابلیت نصب روی موبایل و دسکتاپ.

گیمیفیکیشن:

امتیازدهی پیشرفته، Leaderboard، مدال‌های شخصی، سیستم جوایز، ماموریت‌های روزانه.

مدیریت محتوا: سیستم هوشمند انتشار مقالات، تست‌های روانشناسی، راهنماها و محتواهای سلامت.

واکنش‌گرا: کاملاً ریسپانسیو؛ بهینه برای موبایل و تبلت.

امنیت: JWT، rate limiting، password hashing، monitoring لاگ‌ها.

مقیاس‌پذیری: آماده برای چند زبانه شدن (i18n)، توسعه ماژول‌های جدید، رشد حجم کاربران.

ابزارهای مورد استفاده: GitHub برای کنترل نسخه، Vercel یا Render برای دیپلوی سریع اولیه، Upstash برای Redis.

ویژگی خلاقانه: تحلیل رفتار کاربران و پیشنهاد خودکار سرویس‌های سلامت بهتر.

دیزاین: تم‌های قابل تعویض، Light/Dark، انیمیشن Splash، Skeleton Loading در صفحات اصلی.

API اکوسیستم: اتصال به سرویس‌های روانشناسی، مشاوره تغذیه، ابزارهای تحلیل HR خارجی (در آینده).

مدیریت پروژه: راه‌اندازی ساختار CI/CD ساده با GitHub Actions.

مراحل اولیه:

ساخت MVP با ۵ ماژول اصلی

تست داخلی (کارمندان پرانا)

توسعه فیچرهای هوشمند و گیمیفیکیشن

فاز عمومی (سازمان‌ها و افراد)
